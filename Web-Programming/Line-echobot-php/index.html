開發環境

  • PHP 5.4以上
  • Ubuntu 16.04
  • Composer 1.2.2

Line Bot帳號設定

首先，請先到 Line Business Center 註冊 BOT 帳號，並且啟用Messaging API服務。

進到 Line Manager 頁面中，選擇剛註冊的BOT帳號，至帳號設定->Bot設定。
