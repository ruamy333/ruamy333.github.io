開發平台

  • windows 10
  • xampp 7.0.15 portable版
  • PHP 7.0.15
  • MS SQL Server 2012

XAMPP安裝

XAMPP比較特別的是提供三種版本

  • 安裝版本（installer.exe）。
  • 免安裝（xampp-win32-x.x.xx-x-VCxx.zip）版本。
  • portable版：portable版移除FileZilla FTP伺服器與Mercury郵件伺服器等等的工具，所以XAMPP檔案體積也比較小。

此處我們使用portable版本，只需要執行XAMPP的『setup_xampp』批次檔來更新環境中的路徑，不需要自己手動設定。
XAMPP官網下載

選擇「更多下載」

選擇作業系統與PHP版本後，選擇要下載的版本，此處下載portable版。

解壓縮後執行setup_xampp.bat即可以開始使用xampp。

設定 PHP 開發的開發環境

PHP的 php_mssql.dll是用來與SQL Server 2000連線，因此若是使用SQL Server 2005以上版本，就要用到微軟為PHP提供的擴充功能（SQL Server Driver for PHP）。
