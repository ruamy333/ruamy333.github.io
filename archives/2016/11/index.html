使用 PHP 建立 Line Bot

15 Nov 2016

一、開發平台：

  • PHP 5.4以上
  • Ubuntu 16.04
  • Composer version 1.2.2

二、Line Bot帳號設定

1.首先，請先到 Line Business Center 註冊 BOT 帳號，並且啟用Messaging API服務。

2.進到Line Manager 頁面中，選擇剛註冊的BOT帳號，至帳號設定->Bot設定。

3.確認身分是否為LINE Developers，Reply message APIæ¯否可用，並將webhook傳訊功能開啟。

4.至LINE developers 中，加入Web Server IP。

5.以下是傳訊時會需要的資料，Channel Secret要點SHOW才會出現。

6.Webhook URL則要改成Line bot回應的程式檔案，因為Line 要求一定要SSL憑證，因此此處的網址需要用https，且一定要使用Domain
Channel Access Token是傳訊時會需要的資料。

 

三、Web Server設定

1.因為Line 要求一定要SSL憑證，因此要在Server上安裝SSL憑證，在此處我們用Let’s Encrypt這家免費SSL商。

2.下載官方自動安裝憑證程式。

3.進入剛剛下載的資料夾。

4.將網域輸入。

5.設定過程中會需要輸入信箱，結束後程式會幫你自動加載SSL資料進conf，所以無需做任何設定。

 

四、Line SDK Install

1.使用Line bot API需要先下載line-bot-sdk，而line-bot-sdk需要使用Composer來安裝，因此要先安裝Composer。

2.下載並安裝Composer。

五、建立LINE Bot

  1. 建立Echo Bot的Class，命名為EchoBot.php，實作5個方法。
